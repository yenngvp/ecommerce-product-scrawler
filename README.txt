# Project: ProductZ project

# Features
* Fast
* Resource efficient
* Building spiders for crawling products from top e-commerce web sites in Vietnam.
Then the massive product data will be used to build a social network app for consumers, shoppers to find any selling product information, do price comparison and ask other past buyers for consulting buying decision and criteria.
